TABAQ Rev207 — Commerce setup This package includes the static TABAQ storefront plus a local Node backend for cart validation, order storage, and Stripe Checkout session creation. Run locally: 1. Install Node.js 20 or newer. 2. Open a terminal in this folder. 3. Set your Stripe key: - macOS/Linux: export STRIPE_SECRET_KEY=sk_test_your_key_here - Windows PowerShell: $env:STRIPE_SECRET_KEY="sk_test_your_key_here" 4. Optional: set STRIPE_WEBHOOK_SECRET and ADMIN_TOKEN. 5. Run: npm start 6. Open: http://localhost:8787 Useful endpoints: - GET /health - GET /api/catalog - POST /api/cart/validate - POST /api/checkout/session - POST /api/webhooks/stripe - GET /api/orders/:order_id - GET /api/admin/orders requires Authorization: Bearer ADMIN_TOKEN Notes: - Opening index.html directly with file:// will still show the site and cart, but payment checkout requires the backend server. - The backend stores local development orders in backend/data/orders.json. - Do not commit or share live Stripe secret keys.
